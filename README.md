# 🔐 图片加密解密工具 (Image Encryptor Tool)

这是一个完全在浏览器中运行的图片加密解密工具。

您可以使用它和自定义密码来加密您的私人图片。所有操作都在您的本地电脑或手机上完成，图片文件**永远不会**被上传到任何服务器，100% 保护您的隐私。

## ✨ 主要功能

* **纯客户端运行**：所有加密和解密操作均在您的浏览器中完成，无需联网，隐私绝对安全。

## 📖 如何使用

### 加密图片

1.  访问上面的“快速访问”链接。
2.  点击或拖拽一张**普通图片**到上传区域。
3.  在“加密/解密 密码”框中输入一个您自己的密码（请牢记！）。
4.  点击 **🔒 加密** 按钮。
5.  右侧“处理后图片”会显示为一张雪花图（即加密形态）。
6.  点击 **💾 下载图片** 按钮，保存加密后的图片。

### 解密图片

1.  刷新页面，或重新打开工具。
2.  点击或拖拽一张**加密后**的图片（雪花图）到上传区域。
3.  工具会自动提示“检测到加密图片”。
4.  在密码框中输入您之前加密时使用的**相同**密码。
5.  点击 **🔓 解密** 按钮。
6.  右侧“处理后图片”会还原为您加密前的原始图片。

## ⚠️ 重要提示：关于安全

1.  **隐私安全**：
    本工具是**纯前端**应用。您的图片和密码**永远不会**离开您的电脑或手机，不会被发送到任何服务器。

2.  **加密强度**：
    本工具提供的是一种“轻量级”加密（像素位置打乱 + 颜色异或），主要用于防止他人**随意查看**您的图片。它**不是**军事级或金融级加密 (如 AES)，请勿用于极端机密的场景。

3.  **密码遗忘**：
    请务必记住您的密码！**如果密码丢失，加密后的图片将永远无法恢复！**（因为解密算法完全依赖于这个密码）。

## 💡 工作原理

本工具的加密过程主要分为两步，解密是其完全的逆向操作：

1.  **密码哈希 (生成种子)**：
    使用一个确定性哈希函数 (`hashPassword`) 将用户输入的任意密码字符串转换为一个固定的32位整数（种子）。

2.  **像素位置打乱 (三重)**：
    使用这个“种子”作为伪随机数生成器的基础，对图片的所有像素位置进行三次可复现的随机打乱（基于 Fisher-Yates 算法变体）。

3.  **像素颜色加密**：
    对打乱后的每个像素的 RGB 值，再次使用“种子”结合像素位置，进行强力的异或 (XOR) 和加法运算，使其颜色完全失真。
